# chrome win

## dev console

- open developer console: `ctrl` + `shift` + j

## tab control

- search in current tab: `ctrl` + l
- search in new tab: `ctrl` + t
- close tab: `ctrl` + w

- switch to n-th tab: `ctrl` + #
- swtich to prev/next tab: `tab` + `ctrl` + (optional: `shift`)

- back/forth on tab history: `alt` + `arrow-left` / `arrow-right`
