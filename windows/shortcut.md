# shortcut

## Win Key

- Win + D: show desktop
- Win + E: open `File Explorer`
- Win + S: open Cortana search
- Wiin + L: lock computer

### switch desktop

- Win + tab: Manage desktop
- Win + Ctrl + D: create new desktop
- Win + Ctrl + left/right arrow: Switch desktop
- Win + Ctrl + F4: Delete current desktop
